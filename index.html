Events calendar

Events calendar demo app built with AngularJS

Supports the following features:

  • User will be presented with a full page calendar for the current month and year. User can select a date and set a reminder/note for the selected day.
  • User can select a date and delete any of the reminders and notes for the selected day.
  • User can set an infinite number of reminders/notes for a given date.
  • Reminders and notes should be persistent between user sessions.
